Generate CV

Create a local targeted CV draft

The model uses the canonical profile, deterministic match context, and the pasted job to draft a stronger Markdown CV. Configure your local LLM API key in .env with JMI_LLM_API_KEY. You can also set a default provider/model there with JMI_LLM_PROVIDER and JMI_LLM_MODEL. Base URL is optional and only needed for custom gateways.
